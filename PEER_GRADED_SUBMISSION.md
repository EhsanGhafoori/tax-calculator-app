## Project Features Checklist

### ✅ GitHub (6 points)
- ✅ Public GitHub repository URL
- ✅ Redux-related files and code implemented correctly

### ✅ Landing Page (5 points)
- ✅ Background image displayed properly
- ✅ Paragraph about the company
- ✅ Company name displayed
- ✅ Get Started button linking to product listing page

### ✅ Product Listing Page (9 points)
- ✅ Six unique houseplants for sale, each displaying thumbnail, name, and price
- ✅ Plants grouped into at least three categories
- ✅ Add to Cart button for each plant:
  - ✅ Shopping cart icon increases by one after clicking
  - ✅ Button becomes disabled after clicking
  - ✅ Appropriate plant gets added to shopping cart

### ✅ Header (7 points)
- ✅ Displays on both product listing page and shopping cart page
- ✅ Shopping cart icon with value displaying total number of items
- ✅ Navigation to either of the other pages

### ✅ Shopping Cart Page (23 points)
- ✅ Total number of plants in the cart
- ✅ Total cost of all items in the cart
- ✅ Each plant type displays thumbnail, name, and unit price
- ✅ Increase button increments items and updates all values
- ✅ Decrease button decrements items and updates all values
- ✅ Delete button
- ✅ Checkout button (displays "Coming Soon" message)
- ✅ Continue shopping button linking to product listing page
